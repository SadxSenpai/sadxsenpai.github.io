# Portfolio Website

Live: [sadxsenpai.github.io](https://sadxsenpai.github.io)

## Tech Stack

Frontend: HTML5, CSS3, JavaScript

Tools: Visual Studio Code, Git, GitHub Pages

## Techniques

CSS Grid and Flexbox for responsive layouts

CSS variables for theming and design consistency

Vanilla JavaScript for interactivity (typewriter effect, navigation, smooth scrolling)

Glassmorphism effects with backdrop filters

Terminal-style UI components

Dark theme with cyberpunk aesthetic

Canvas animation for hero section

Responsive design with mobile-first approach
