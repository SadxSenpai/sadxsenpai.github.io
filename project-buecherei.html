Buecherei Windows App (Library Manager)

Project Overview

A robust desktop application designed to manage library inventories. This project demonstrates proficiency in C#, Windows Forms (WinForms), and CRUD data operations, mimicking real-world enterprise software requirements.

Note: This represents my first attempt at building a real-world ERP system. As an early learning project, it suffers from architectural flaws such as improper class implementation and lack of code reusability, which I have since learned to avoid.

Key Features

  • Inventory Management: Full Create, Read, Update, Delete (CRUD) functionality for book entries.
  • Search & Filter: Implemented logic to filter the book list by Title, Author, or ISBN in real-time.
  • User Interface: Designed a clean, event-driven Windows interface using standard controls (DataGridView, TextBoxes, Buttons).
  • Data Persistence: Connects to a MySQL database to store library inventory, supporting persistent data management.

Technical Skills

  • Object-Oriented Programming (OOP): Utilized classes to encapsulate data and methods.
  • Event Handling: Managed UI events to create a responsive user experience.
  • Collections: Used Lists and LINQ to handle data sorting and searching efficiently.

How to Run

  1. Download the repository.
  2. Open the .sln file in Visual Studio 2022.
  3. Build and Run (F5).
