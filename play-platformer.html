Funny Platformer (Tech Demo)

Project Overview

A physics-based 2D platformer designed to demonstrate robust game architecture and responsive character controls. This project serves as a Vertical Slice to showcase core Unity development skills.

Note: This project is a "Vertical Slice" tech demo intended to showcase specific mechanics and architecture. It is not a full-length game and focuses on depth of polish over breadth of content.

Desktop Required

This WebGL game is not optimized for mobile devices. Please visit on a desktop computer to play.

Key Features

  • Advanced Movement: Implemented fluid mechanics including Double Jump, Wall Sliding, and Wall Jumping using Rigidbody2D physics.
  • Animation State Machine: Uses a clean Enum-based system to handle animation states (Idle, Run, Jump, Fall) without visual bugs.
  • Save System: Features a persistent High-Score system using local data storage (PlayerPrefs).
  • System Architecture: Includes a dynamic Settings Menu that automatically detects and applies valid screen resolutions for the user's monitor.

Technical Skills

  • Physics: Physics2D.OverlapCircle for ground checks, Velocity clamping for wall slides.
  • Patterns: Singleton Pattern (ScoreManager) for easy data access.
  • Audio: Persistent background music that seamlessly transitions between scenes using DontDestroyOnLoad.

How to Play

  • Move: A / D or Arrow Keys
  • Jump: Spacebar (Press again for Double Jump)
  • Wall Jump: Press Jump while sliding down a wall.
